Installation guide:
- Activate Open4All patch
- Copy everything from Z:\resource\java\jvm\lib\jrt to C:\resource\java\jvm\lib\jrt (skip this if you have done this earlier)
- Copy javapiglerapi.odc at C:\resource\java\jvm\lib\jrt
- Copy javapiglerapi.jar at C:\resource\java\jvm\lib\jrt\odc
- Copy piglerjavaapi.dll at C:\sys\bin
- Create midpodclist at C:\resource\java if it doesn't exist
- Add "javapiglerapi.odc" and leave 2 empty lines (!) in the end of midpodclist file
